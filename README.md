# 家庭圈用户识别模型

## 项目简介

本项目实现了基于多维度数据的家庭圈用户识别模型，能够准确界定家庭范围，识别家庭成员，并确定家庭关键人信息。

```
梧桐杯/

│   ├── train_result.csv                    # 训练数据预测结果
```

## 功能特性

### 1. 特征工程
- **地址关联性特征**：基于用户地址信息识别家庭关联
- **缴费账户关联性特征**：通过缴费账户识别家庭成员关系
- **终端共享特征**：基于设备共享关系识别家庭圈
- **通信行为特征**：通过通话记录和联系人信息识别关系
- **基础信息特征**：用户性别、年龄等基础特征

### 2. 家庭圈识别算法
- 使用DBSCAN聚类算法进行家庭圈识别
- 基于多维度特征进行用户聚类
- 自动识别家庭圈边界

### 3. 关键人识别
- 基于账户主特征识别关键人
- 考虑缴费行为、通信中心度等因素
- 为每个家庭圈确定关键人

### 4. 关系判定规则
- 地址关联规则（权重0.3）
- 账户关联规则（权重0.4）
- 设备共享规则（权重0.2）
- 通信行为规则（权重0.1）

### 5. 可视化界面
- 统计信息展示
- 家庭圈列表浏览
- 用户信息查询
- 关系查询功能
- 数据可视化图表

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 运行主程序

```bash
python main.py
```

这将：
- 加载Excel数据文件
- 进行特征工程
- 训练模型并识别家庭圈
- 识别关键人
- 保存结果到output目录

### 2. 启动Web可视化界面

```bash
python app.py
```

然后在浏览器中访问：http://127.0.0.1:5000

## 数据格式

Excel文件包含3个sheet：
- **Sheet 1**: 带标签的训练数据（5000条）
- **Sheet 2**: 不带标签的验证数据（5000条）
- **Sheet 3**: 不带标签的测试数据（2000条）

每个sheet的格式：
- 第一行：英文表头
- 第二行：中文表头
- 第三行开始：数据

## 输出结果

### 1. CSV结果文件
- `train_result.csv`: 训练数据预测结果
- `valid_result.csv`: 验证数据预测结果
- `test_result.csv`: 测试数据预测结果

每行包含：
- 用户ID
- 家庭圈ID
- 是否关键人（1表示是，0表示否）

### 2. 统计报告
`statistics.json` 包含：
- 各数据集的总用户数
- 识别出的家庭圈数量
- 关键人数量

## 模型评估

模型评估指标包括：
- **准确率 (Accuracy)**
- **精确率 (Precision)**
- **召回率 (Recall)**
- **F1值 (F1-Score)**

目标：核心关系识别准确率 ≥ 85%

## 技术特点

1. **多维度特征融合**：综合地址、账户、设备、通信等多维度信息
2. **无监督学习**：使用DBSCAN聚类算法，无需大量标注数据
3. **规则与模型结合**：结合规则判定和机器学习模型
4. **可解释性强**：提供关系判定规则和置信度
5. **交互式可视化**：提供友好的Web界面进行结果展示和查询
